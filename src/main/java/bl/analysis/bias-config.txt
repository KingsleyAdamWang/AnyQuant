【1】乖离率（BIAS）简称Y值也叫偏离率，是反映一定时期内股价与其移动平均数偏离程度的指标。移动平均数一般可视为某一时期内买卖双方都能接受的均衡价格。因此，股价距离移动平均线太远时会重新向平均线靠拢。乖离率指标就是通过测算股价在波动过程中与移动平均线出现的偏离程度，从而得出股价在剧烈波动时因偏离移动平均趋势可能形成的回档或反弹。
乖离率达＋5％以上为超买现象，是卖出时机。短期获利较大，获利回吐的可能性较高。
乖离率达-5％以下为超卖现象，是买入时机。空头回补的可能性较高。
在趋势上升阶段股价出现负乖离，正是逢低买入的有利时机。
在趋势下降阶段股价出现正乖离，正是逢反弹出货的最佳时机。
乖离率在零值附近徘徊，且与股价走势呈相同趋势，股价小幅波动，比较稳定。应静观其变，不宜轻举妄动。
